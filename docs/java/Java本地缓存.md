## 缓存框架
### [JCache](https://github.com/jsr107/jsr107spec)

JCache 是 Java 缓存 API。它由 JSR107 定义。它定义了一个供开发人员使用的标准 Java 缓存 API 和一个供实施者使用的标准 SPI。

Java Caching API 为 Java 程序从缓存中创建、访问、更新和删除条目提供了一种通用方法。
